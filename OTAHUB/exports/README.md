## Adobe Illustrator (.ai)
SVG files are fully compatible with Adobe Illustrator.
To convert: File > Open > select .svg → then File > Save As > Adobe Illustrator (.ai)

The SVG has two named groups:
- **#o-ring** — cyan O outline
- **#torii-gate** — pink torii gate
